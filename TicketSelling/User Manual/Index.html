Entry Form

Admin

What is admin?

    Admin is person who is in charge of operation of computers, website and arrangements and tasks needed to control the organization or plans. There are 3 admin roles in my system.
They are MD, Manager, Staff.
    MD has full access to all system with permissions. Manager has fewer access then MD but Staff has restricted access to system.
    In the left side of my software, there is entry button in the navigation. When you click it, drop down bar is just show up and you can see Admin and user buttons. When you click admin button, you can see form as the picture on the right side. There are several textboxes, photo and combo box to fill admin's information.

You must fiil all the textbox according to the format.
1. Photo (You must add jpg, png or jpeg format of admin's passport picture)
2. Role (Level of your Employee to permit the system)
3. Name (Real name)
4. Username (This is use to login the system. Username are all lowercase and between 3 and 10 words)
5. Password (Password are between 8 to 16 characters)
6. Gmail
7. Phone Number
8. NRC (Nation ID card. You must write according to the format. Eg. 14/HATHATA(N)111111)
9. Address ( Your current address)
10. City (Your birth city)
11. Postcode /Zip (Your city code)
  If you complete filling the textboxes, you can click save button to save the admin information.

Update your information

    If you want to change your information or your employees' information, you need to double click the row that you want to change, all the information will appear in the textboxes. Then, you can change whatever you want. After you complete changing, you need to click update button to save. Successful message will show up and update successful.

Delete your information

    If you want to delete admin's information, you need to click the trash bin with red color on the right size of the information you want to delete. And then the confirmation box will show up. If you want to continue, press Ok but if you want to cancel, just press cancel.

Reminders : You cannot delete if you have only one MD level.

SEATS

Seats

There are 2 types in system. Seat Type & Seat


Seat Type
    Seat Type is used to name the seat row in cinema. Eg. A,B,C. In this system, you must fill seat type and note. Note is used to note something you need to remember about seat row of price and something else. If you want don't want to note, add - and click save. Seat Type and Seat Configuration are avaiable only for MD level.

Seat Type location
Configuration/Seats/Seat Type

Update Seat Type

    If you want to change seat type or note, you need to double click the row that you want to change, all the information will appear in the textboxes. Then, you can change whatever you want. After you complete changing, you need to click update button to save. Successful message will show up and update successful.

Reminder : You cannot update seat type when you have inserted seat name and price in Seat Form.

Delete Seat Type

    If you want to delete seat type, you need to click the trash bin with red color on the right size of the information you want to delete. And then the confirmation box will show up. If you want to continue, press Ok but if you want to cancel, just press cancel.

Reminders : You cannot delete when you have inserted seat name and price in Seat Form.

Seats

    Seat Form is used to apply seat price and seat count according to the seat type. In seat form, you need to choose seat type name. Seat type name is appeared only you assigned in the seat type form. And you need to fill seat count and seat price. Seat form is avaiable only for MD level.

Seat Form location
Configuration/Seats/Seat

Reminder : You must set equal seat count for all seat.

Update Seat
    you need to double click the row that you want to change, all the information will appear in the textboxes. Then you can change. AFter you finish, press update.
Delete Seat
    you need to click the trash bin with red color on the right size of the information you want to delete. And then the confirmation box will show up. If you want to continue, press Ok but if you want to cancel, just press cancel.
    You cannot update or delete the seats price or count or name when you are on the stage of selling ticket. You must contact developer team to maintain the software and data to modify your cinema.
